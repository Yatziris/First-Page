Html & Css Cheatsheet

Basic Html Page Layout

Tag Description
table The wrapper element for all HTML tables
thead The set of rows defining the column headers in a table
th Defines a header cell in an HTML table
tbody The set of rows containing actual table data
tr The table row container
td The table row data container
tfoot The set of rows defining the footer in a table

Html Commands

Tag Description
h1 Establishes a heading. Can be h1 through h6.
p Establishes a paragraph. There is an automatic line break after the closing tag.
br Creates a line break. This is a self-closing tag.
img Displays an image on the webpage.
form Sets up interface to collect and submit user supplied information.
Input type='text' name='username' Creates an interactive element allowing user to make input. Can use 'radio', 'checkbox', 'text' for example. Y0u must include a "type= "and a "name= ". It has no closing tag.
em Emphasizes text, browsers usually render as italics.
ol Creates an orderd list. List items will be numbered.
ul Creates an unordered list. Items will have bullets.
li The line tag is used to add items to a list. It is used in both ordered and unorderd lists.
"href="url">'text desciption of link'" This is the 'anchor' element that creates either an internal or external link.

CSS Commands

Property Name: Value Affect
font-family: Monoscape, "Times New Roman, serif; This will change the targeted font. It will attempt the fonts in order. Multiple fonts are listed in case the browser doesn't support the selected font.
font-size: 30px; Changes the targed item's font to whatver value is after the colon.In this case 30px.
color:red; Changes the font color to red.
background-color: blue Makes the background of the selected area the color selected.
font-weight: bold Sets how bold the text is. It can be light, normal, bold, etc. Or, it can be a number between 100-900 for extra control.
text-transform: value Transforms the font to the given value. Values are none, uppercase, lowercase, capitalize, full-width.
text-align: Allows you to justify the text left, right, center, or justify.
